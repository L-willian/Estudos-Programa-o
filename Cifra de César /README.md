# 🔒 Cifra de César

Implementação de um criptógrafo usando Cifra de César em **C**.

## 📋 Objetivo do Projeto

Este projeto implementa uma criptografia clássica baseada em:
- **Cifra de César**: Deslocar cada letra por um número fixo de posições no alfabeto
- Criptografia e descriptografia de mensagens
- Demonstrar conceitos de criptografia simples

É um projeto educacional para entender os conceitos básicos de criptografia e manipulação de strings em C.
